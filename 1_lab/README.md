# itmo-containers
## Плохой доцкер файл

1) Используется тег `latest`
2) Всё делается из-под root пользователя 
3) Очень много слоев, образ весит много
4) Отсутствует EXPOSE 

## Хороший доцкер файл

1) Устанавливается определенная версия (5)
2) Установка пакетов под root, дальше идет переключение под user-1000
3) Слои скомпанованы
4) Установка библиотек вынесенный в отдельный файл
5) EXPOSE присутствует
